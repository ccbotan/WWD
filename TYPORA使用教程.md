# TYPORA 使用教程

| 纲要                    |      |                                |
| ----------------------- | ---- | ------------------------------ |
| Ctrl + Shift + 1        |      | 打开大纲                       |
| Ctrl + Shift + 2        |      | 打开文件                       |
|                         |      |                                |
| 字体                    |      |                                |
| Ctrl + B                |      | 加粗                           |
| Ctrl + I                |      | 倾斜                           |
|                         |      |                                |
| ```                     |      | 代码框                         |
| ```mermaid              |      | mermaid语言的代码框            |
| ctrl  + T               |      | 插入表格                       |
| ctrl  + Shift + Q       |      | 插入引用                       |
|                         |      |                                |
| Ctrl + 1                |      | 大标题                         |
| Ctrl + 2                |      | 二级标题                       |
| Ctrl + 3                |      | 三级标题                       |
| ....                    |      |                                |
|                         |      |                                |
| 数学符号                |      |                                |
| $$ + Enter              |      | Latex 输入框                   |
| 2 * $   $text\ content$ |      | $\frac{1}{m} + |\overline{A}|$ |
|                         |      |                                |
| 选中                    |      |                                |
| Ctrl + A                |      | 全选                           |
| Ctrl + L                |      | 选中当前行或句                 |
| Ctrl + D                |      | 选中当前词                     |



* 目录需要在最开头的地方输入 [TOC] 并回车
* 可以在主题更改主题颜色
* 视图下 有个 功能 **保持在窗口最前端** 也好用



如果遇到某个数学符号不会打, 问豆包"Latex 中 {某数学符号} 怎么打"
$$
3^6 + \overset{.}{f}(k) = \sum_{i = 1}^k k^2
